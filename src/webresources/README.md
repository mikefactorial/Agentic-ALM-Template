# src/webresources

TypeScript web resources — form scripts, ribbon handlers, and custom client-side logic for model-driven apps. One subfolder per solution area.

## What Lives Here

```
src/webresources/
  acm_AcmePlatform/
    WR-AccountForm/
      src/
        main.ts          # Entry point — exports namespace object (becomes window.{Name})
      dist/              # Compiled IIFE bundle (generated by npm run build)
      vite.config.ts     # Must use IIFE library mode with no content hashing
      package.json
      tsconfig.json
```

## What Web Resources Are

Web resources are TypeScript files compiled to a single JavaScript bundle and deployed to Dataverse. They run in the browser context of a model-driven app and interact with the platform via the Xrm client API.

Common uses:
- Form scripts registered on OnLoad, OnChange, and OnSave events
- Ribbon command bar button handlers
- Lightweight client-side automation that doesn't warrant a full PCF control

## When to Use

| Need | Use |
|------|-----|
| Form/ribbon script logic | Web resource (this folder) |
| Custom UI component on a model-driven form | PCF control (`src/controls/`) |
| Standalone app with connectors and rich UI | Code app (`src/codeapps/`) |
